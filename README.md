# GridLogic - Test de Aptitud Visual-Espacial

## Descripción del proyecto
**GridLogic** es una aplicación móvil desarrollada en Android Studio y diseñada para evaluar la capacidad de razonamiento lógico y la habilidad visual-espacial de los usuarios. El proyecto está inspirado en el **Test de matrices progresivas de Raven**, el cual es un estándar en la medición de la inteligencia no verbal.

### Características principales:
* **Interfaz Moderna:** Desarrollada con Jetpack Compose.
* **Navegación fluida:** Implementación de un grafo de navegación tipado.
* **Interactividad:** Selección de manera dinámica de opciones con retroalimentación visual.
* **Resultados:** Pantalla de cierre que valida la finalización de todo el proceso.

## Tecnologías utilizadas
* **Lenguaje:** Kotlin
* **Framework UI:** Jetpack Compose
* **Componentes:** Material Design 3
* **Control de Versiones:** Git y Github

## Instrucciones para ejecutar la app
Para ejecutar y utilizar el proyecto en tu entorno local, sigue estos 3 pasos:

1. **Clonar el repositorio dentro de Android Studio.**
2. **Ejecuta la app con el emulador, o bien con un dispositivo móvil físico.**
3. **Una vez ejecutada la app, lee las instrucciones de la pantalla y presiona el botón para empezar el test.**
